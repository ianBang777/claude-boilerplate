# Support Tech Reaction

슬랙 리액션 이벤트를 처리하여 Notion 태스크를 업데이트한다.

## 입력 형식

```
type: reaction
이모지: {이모지 이름}
반응한 사람: {이름}
메시지 ID: {슬랙 메시지 타임스탬프}
```

## 동작

### 1단계: Notion 태스크 조회

Notion 커넥터로 Support Tech 데이터베이스에서 `메시지 ID`와 일치하는 항목을 찾는다.

- 일치하는 항목이 없으면 종료한다.

### 2단계: 이모지 유형별 처리

#### `heavy_check_mark` — 담당자 지정

| 필드 | 변경값 |
|------|--------|
| 담당자 | 반응한 사람 이름 |
| 완료 여부 | 변경 없음 |

#### `done` — 완료 처리

| 필드 | 변경값 |
|------|--------|
| 완료 여부 | 완료 (true) |
| 담당자 | 반응한 사람 이름 (담당자가 비어있는 경우에만) |

## 주의사항

- 위 두 이모지 외의 이모지는 무시하고 종료한다.
- 이미 완료 처리된 태스크에 `done` 리액션이 오면 업데이트하지 않고 종료한다.
- Notion 업데이트 실패 시 오류를 로그로 남긴다.
